# Sparzas
Ecommerce
1. O Coração: Modelo de Dados (Backend)
Primeiro,  definir como um produto se parece.  Mongoose (MongoDB) para este exemplo por ser flexível.


2. O Cérebro: Rotas da API (Backend), caminhos para o frontend buscar esses produtos.


3. A Vitrine: Componente de Produto (Frontend)
No React, card para exibir esses dados de forma elegante.


4. O Carrinho: Lógica de Estado (Frontend)
Para gerenciar o que o usuário quer comprar, usamos o useState do React.
